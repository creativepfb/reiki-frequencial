# Reiki Frequencial

App web de terapia vibracional com as frequências Hz dos símbolos do Reiki Usui e Karuna Ki.

## Funcionalidades

- Busca por sintoma ou benefício
- Filtro por sistema (Reiki Usui / Karuna Ki)
- Player de áudio com barra de progresso
- Modo Playlist Completa (toca todos em sequência)
- 15 símbolos com informações de chakra, região e benefícios

## Configuração

Antes de publicar, edite a variável `BASE_URL` no `index.html`:

```js
const BASE_URL = 'https://SEU-DOMINIO.com.br/audios/';
```

Suba os arquivos `.mp3` na pasta `/audios/` do servidor.
